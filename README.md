Ruby wrapper to get configured with the commenting engine, [Komments][]

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'komments'
```

And then execute:

```shell
bundle
```

Or install it yourself as:

```shell
gem install komments
```
